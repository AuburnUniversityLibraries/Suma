About

Suma is set of free software and tools that allows for observational data about space usage to be gathered on a tablet device (such as an iPad), then analyzed and visualized in a browser-based interface. Suma streamlines existing data collection methods, enabling quick, hassle-free mobile data collection and storage. Suma also provides a sophisticated suite of data analysis and visualization capabilities, including a set of charts that dynamically recombine and display data based on the user's input parameters, and exportable spreadsheets and graphs. Suma is an open-source software project from North Carolina State University Libraries, created with the goal of promoting observational data analysis as an integral part of service / space design and day-to-day planning.

The case for Suma

Many libraries produce manual headcounts or activity counts for physical spaces, but struggle with the difficulties of collecting, organizing, storing, and analyzing such data, significantly limiting its use. The availability of tablet devices has created an opportunity to simplify and encourage the collection of fine-grained data about the use of library services and physical spaces.
